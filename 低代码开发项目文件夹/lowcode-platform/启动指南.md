## 🚀 快速启动

### 1. 安装依赖
```bash
# 进入前端目录
cd frontend

# 安装依赖（推荐使用国内镜像源）
npm config set registry https://registry.npmmirror.com
npm install
```

### 2. 启动开发服务器
```bash
# 启动前端开发服务器
npm run dev
```

### 3. 打开浏览器
```
访问地址：http://localhost:3000
```

### 4. 一键启动（Windows）
```bash
# 双击运行根目录下的启动脚本
启动.bat
```


## 🛠️ 技术栈

### 前端
- **Vue 3.3** - 渐进式JavaScript框架

- **Vite** - 现代化构建工具
